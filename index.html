How the product works

The business owner fills out a simple setup form. The system uses those answers to generate:

  • a chatbot personality and goal set
  • starter FAQ knowledge
  • lead capture rules
  • a website widget preview
  • an embeddable script snippet

In a full version, this would save their settings to a database and connect to an AI API.

Best MVP flow

  1. User enters business information
  2. Page generates a chatbot config JSON
  3. User tests a live preview
  4. User copies website embed code
  5. Backend serves chatbot responses on their live site
